API DOCUMENTATION

V1.0 BASE URL https://api-ml-sdg.azurewebsites.net


This ML API contains one endpoint that is

  • /api/getpredictions

Curl

curl --location --request POST 'https://api-ml-sdg.azurewebsites.net/api/getpredictions' \

--header 'Authorization: Token unique_token' \

--header 'Content-Type: application/json' \

--data-raw '{"text":"TitleAbstract_text",

"unique_id":"EID"}'

NOTE : In this example we using the unique Id as EID

Python - Requests

import requests

import json

url = "https://api-ml-sdg.azurewebsites.net/api/getpredictions"

payload = json.dumps({

"text": "TitleAbstract_text",

"unique_id": "EID"

})

headers = {

# add unique token below

'Authorization': 'Token unique_token',

'Content-Type': 'application/json'

}

response = requests.request("POST", url, headers=headers, data=payload)

print(response.text)

NOTE : In this example we using the unique Id as EID
